# SEAL STORAGE

## 概要
フレークシールの管理システムです。
シールの絵柄やタグの登録、更新、削除ができ、一覧の表示や検索機能が使えます。

## 主な機能
- ログイン、ログアウト
- ログインユーザーが登録したシールの一覧表示
- パッケージ名の登録、更新、削除
- シール名、画像、タグの登録、更新、削除
- シール名、タグでの検索
